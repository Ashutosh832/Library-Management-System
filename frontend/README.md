# Frontend

React frontend for Library Management System.

## Technologies

- React
- React Router
- Fetch API
- CSS

## Installation

npm install

## Run

npm start

## Features

### Authentication

- Login
- Signup
- Logout

### Student Dashboard

- View Books
- Search Books
- Issue Books
- Return Books
- View My Issues

### Admin Dashboard

- View Active Issues
- View Logs
- Manage Users
- Manage Books
